Titre : Laboratoire 1 - Utilisation DirectShow
Date : 2019-09-10

Procédure d'installation
1- Créer un projet c++ vide dans VisualStudio
2- Inclure le fichier main.cpp dans les fichiers sources du projet
3- Placer la video "Exemple.avi" dans la racine C:\
4- Appuyer sur F5 pour compiler

Procédure d'utilisation
1- Après la compilation, deux fenêtres vont s'ouvrir. 
	- La première est la vidéo "Exemple.avi"
	- La console de commande
2- Vous devez avoir la console de sélectionner pour faire les manipulations suivantes
	- P : Lecture ou Pause
	- A : Avance Rapide
		- Pour annuler l'avance rapide, appuyer de nouveau sur A ou P
	- R : Retour au Début
	- Q : Quitter
